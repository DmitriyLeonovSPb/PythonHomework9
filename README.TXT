1 Нахождение корней квадратного уравнения находится в файле roots_of_quation.py
При наличие 2 корней формируется список из двух чисел, при корне "0" возвращается список 
вида [0, 0], при отсутствии вещественных корней возвращается список вида [null, null]

2 Генерация csv файла с тремя случайными числами в каждой строке. 100-1000 строк
находится в файле random_generator_csv.py в функции list_to_csv

3 Декоратор, запускающий функцию нахождения корней квадратного уравнения
с каждой тройкой чисел из csv файла и Декоратор, сохраняющий переданные 
параметры и результаты работы функции в json файл объединены в файле func_decoration.py

4 bufer.txt - файл проверки генерации, в дальнейшем коде не участвует,
base.csv - результаты генерации "троек" случайных чисел,
base.json - результаты нахождения корней с помощью функции
